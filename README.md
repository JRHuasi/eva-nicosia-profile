# Vanilla SPA

A simple example of a Single Page Application using Javascript, HTML, and CSS.

Tools used in this example are:

- Webpack
- Sass

# Installation

First install dependencies

```shell
npm install
```

To run the example in development mode

```shell
npm start
```

To build the project

```shell
npm run build
```
